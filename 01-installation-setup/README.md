# Модуль №1 — Установка и настройка BAS

> Первый модуль курса по **BrowserAutomationStudio (BAS)**. Знакомимся с программой, скачиваем и устанавливаем BAS, делаем первый обзор окон и инструментов. Теоретико-вводный модуль — без `.xml`-проекта.

**Уроков:** 3 · **Уровень:** с нуля / для новичков

[← Ко всем темам](../README.md)

## Уроки

1. **Введение в BAS** — что это за программа, её возможности, отличия платной и бесплатной версий.
2. **Установка BAS** — где скачать, какую версию выбрать, установка и первый запуск.
3. **Общий обзор программы** — краткий обзор окон и основных инструментов.
