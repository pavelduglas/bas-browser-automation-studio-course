# Модуль №2 — Интерфейс BAS

> Второй модуль курса по **BrowserAutomationStudio (BAS)**. Подробно разбираем интерфейс программы — внешний и внутренний, все панели и их компоненты. Теоретический модуль — без `.xml`-проекта.

**Уроков:** 6 · **Уровень:** с нуля / для новичков

[← Ко всем темам](../README.md)

## Уроки

1. **Знакомство с внешним интерфейсом** — что видим при запуске и как всё устроено.
2. **Главные элементы внешнего интерфейса** — кнопки, меню и панели.
3. **Знакомство с внутренним интерфейсом** — рабочее пространство для скриптов.
4. **Панель модулей и её компоненты** — из чего состоит и как собирать сценарии.
5. **Панель предпросмотра браузера** — контроль выполнения скрипта в реальном времени.
6. **Панель ресурсов, логов и результатов** — данные скрипта, чтение логов и итогов.
