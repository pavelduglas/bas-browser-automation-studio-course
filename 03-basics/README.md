# Модуль №3 — Основы работы с BAS

> Третий модуль курса по **BrowserAutomationStudio (BAS)**. Осваиваем основы работы с программой: проекты, горячие клавиши и кубики, потоки и режимы, базовую оптимизацию. Практико-вводный модуль — без `.xml`-проекта.

**Уроков:** 4 · **Уровень:** с нуля / для новичков

[← Ко всем темам](../README.md)

## Уроки

1. **Работа с проектами в BAS** — создание, сохранение, открытие и организация проектов.
2. **Горячие клавиши и взаимодействие с кубиками** — приёмы быстрой сборки скриптов.
3. **Потоки и режимы работы** — запуск в несколько потоков и режимы выполнения.
4. **Базовая настройка производительности** — настройки, ускоряющие работу скриптов.
