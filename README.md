# BrowserAutomationStudio (BAS) — бесплатный курс

> Бесплатный курс по **BrowserAutomationStudio (BAS)**: видеоуроки и готовые `.xml`-проекты. Браузерная автоматизация, парсинг, веб-серверы на Node.js/Express, работа с прокси, куки и профилями. Учитесь бесплатно.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## О курсе

Здесь собраны практические проекты на **BAS** — каждый по своей теме. К каждой теме есть видеоуроки на YouTube и готовый проект `.xml`, который можно открыть в BrowserAutomationStudio и разобрать. Всё **полностью бесплатно** и открыто.

- **Видеоуроки** → [плейлист на YouTube](https://www.youtube.com/playlist?list=PLihorGiCATBCSQWamPi1WO4LClUadoY9z)
- **Проекты** → этот репозиторий, по папке на каждую тему
- **Конспекты с кодом** → в папке каждой темы (`lesson-plan.md`)

**Преподаватель:** Павел Дуглас — IT-эксперт и преподаватель.

## Чему вы научитесь

- Создавать и настраивать **веб-сервер на базе BAS** для обработки запросов.
- Принимать и обрабатывать входящие **HTTP-запросы** от внешних систем.
- Реализовывать **многопоточную** обработку задач.
- Использовать **очереди задач** и управлять их выполнением.
- Строить простые **REST API** для запуска и контроля скриптов.
- Передавать данные между сервером и скриптами BAS.
- Интегрировать BAS с внешними системами (боты, сайты, CRM и др.).
- Работать с **Google Drive API** прямо из BAS.
- Прогревать профили куками, работать с прокси и отпечатками.

## Кому подойдёт

- 💻 **Автоматизаторам** на BAS (боты и скрипты).
- 🧑‍💼 **Фрилансерам и разработчикам**, которые хотят запускать задачи через HTTP и строить API на основе BAS.
- 📈 **Владельцам проектов** для масштабирования обработки данных в реальном времени.
- 🔄 **Интеграторам** (Telegram, CRM, сайты и т.д.).
- 🎓 **Продвинутым пользователям BAS** — переход от скриптов к полноценным веб-сервисам.

## Как открыть проект

1. Установите [BrowserAutomationStudio](https://bablosoft.com/shop/BrowserAutomationStudio).
2. Скачайте `.xml` нужной темы из её папки.
3. В BAS: **Проект → Открыть** → выберите `.xml`.
4. Смотрите соответствующий видеоурок и повторяйте.
