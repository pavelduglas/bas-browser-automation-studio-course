# 🎓 Курсовые работы (BAS)

> Практические задания для отработки навыков Browser Automation Studio. Связано: [[Идеи для заработка]]

## 1. Интернет-магазины: мониторинг скидок
**Тема:** автоматизация отслеживания скидок (Wildberries / Ozon).
**Задача:** проверять цену и наличие по списку, уведомлять при снижении.
**Навыки:** БД, ресурсы, логика, XPATH, Telegram.
**Задание:** скрипт по списку артикулов мониторит и анализирует цены; при изменении шлёт уведомление в Telegram-бот по шаблону:

```
Артикул: 777777777
Цена была: 80
Цена стала: 70
Динамика изменения: -12.5%
```

## 2. Парсинг вакансий
**Тема:** сбор вакансий с hh.ru / rabota.by.
**Задача:** собирать вакансии по ключевому слову в Excel.
**Навыки:** Excel, XPATH, запросы (по желанию), веб-эмуляция браузера.
**Задание:** по ключевому слову из ресурса собирать вакансии с rabota.by [название, требуемый опыт, компания] в Excel. После сбора — выключение до следующего ручного запуска.

## 3. Соцсети: массовые действия
**Тема:** автоматизация лайков, подписок, комментариев по запросам или случайно.
**Задача:** протестировать на фейковом или своём аккаунте.
**Навыки:** XPATH, работа с профилями, циклы.
**Задание:** скрипт заходит в поиск Дзен (https://dzen.ru/explore), выбирает случайную тему, переходит в раздел «Статьи» (https://dzen.ru/topic/bigl?tab=articles), выбирает 1–3 случайных статьи, переходит и ставит лайк. После выполнения — выключение.

## 4. Криптовалюта: сбор курсов
**Тема:** парсинг биржевых курсов с Binance.
**Задача:** логировать данные каждые X минут, отправка в Telegram.
**Навыки:** HTTP-запросы, функции, циклы, Telegram.
**Задание:** по паре из ресурсов получать и мониторить цену с Binance через API. Период — раз в 120 сек (задаётся в ресурсах). Скрипт работает в реальном времени постоянно, не выключается.

## 5. Блоги и медиа: автопостинг видео
**Тема:** автоматизация публикации контента.
**Задача:** публиковать видеоконтент.
**Навыки:** веб-эмуляция браузера, работа с профилями, XPATH, HTTP-запросы.
**Задание:** скрипт из папки видео публикует случайные ролики в ВК Клипы со случайным описанием от ChatGPT. После выполнения — выключение.

## 6. YouTube: сбор данных
**Тема:** парсинг названий, просмотров, лайков по ключевым словам.
**Задача:** сбор данных по выдаче YouTube в таблицу.
**Навыки:** веб-эмуляция браузера, XPATH, Excel.
**Задание:** по ключевому запросу из ресурса заходить на YouTube, брать видео из выдачи с лимитом (например, 10), на каждом собирать метрики и сохранять в Excel: название, ссылка, просмотры, лайки, кол-во комментариев. После выполнения — выключение.

## 7. Регистрация аккаунтов
**Тема:** автоматизация регистрации с подтверждением почты.
**Задача:** массовая регистрация с логированием.
**Навыки:** веб-эмуляция браузера, XPATH, БД, Email-модуль, многопоточность.
**Задание:** многопоточный скрипт регистрирует аккаунты Kwork.ru (роль продавца) на емейлы из списка, генерит случайный пароль, заполняет форму, подтверждает по почте, активирует. Ошибки — завершать красным и логировать; успех — писать в БД. Структура БД: `Email(string), Password(string), Активация(bool)`. После выполнения — выключение.

## 8. Микрозадания: автоматизация на буксах
**Тема:** автоматизация простых заданий (посещения, просмотры, лайки, подписки).
**Задача:** эмуляция поведения для получения бонусов.
**Навыки:** веб-эмуляция браузера, XPATH, профили, многопоточность, решение капчи.
**Задание:** скрипт авторизуется в SocPublic и выполняет задания из раздела посещений. Логировать процесс по каждому сайту, выводить заработок и проверять зачисление на баланс. Вход: логин, пароль, API-ключ капчи (RuCaptcha / Captcha Guru). После выполнения — выключение.

## 9. Боты для Telegram Web
**Тема:** взаимодействие с Telegram Web — сообщения, диалоги, подписки.
**Задача:** подписываться на список каналов.
**Навыки:** веб-эмуляция браузера, XPATH, профили, Telegram Web.
**Задание:** скрипт авторизуется в полу-ручном режиме в Telegram Web и подписывается на каналы из txt-списка в ресурсе. Логировать действия, выдавать ошибку при неудаче или лимите. После выполнения — выключение.

## 10. Проверка обратных ссылок (SEO)
**Тема:** проверка наличия заданных ссылок на страницах доноров.
**Задача:** загружать список страниц, проверять наличие нужной ссылки.
**Навыки:** веб-эмуляция браузера, XPATH, регулярные выражения.
**Задание:** скрипт проходит по списку сайтов и проверяет, есть ли ссылка на указанный сайт из ресурсов. Логировать: `сайт — ссылка найдена / не найдена`.
